Quick Start

Fetch provinces first, then request child locations using the selected parent code.

curl "https://location-service-do.vercel.app/api/locations/provinces"
curl "https://location-service-do.vercel.app/api/locations/regencies?province_code=11"

Response Format

All successful API responses use the same envelope.

{
  "log_id": "019ab0f0-c8ec-7d25-9f62-2f23d92fcda3",
  "code": 200,
  "status": true,
  "message": "Success",
  "data": []
}

Recommended Address Form Flow

Use parent codes to load the next level only when users need it. This keeps your UI fast and avoids loading village data too early.

  1. Load provinces on page load.
  2. When a province is selected, call regencies with province_code.
  3. When a regency is selected, call districts with regency_code.
  4. When a district is selected, call villages with district_code.
  5. Save the full_code and name in your app when you need stable references.

Common Use Cases

Address forms

Build province, city, district, and village dropdowns that follow user selection.

Location search

Let users search by location name when they do not know the administrative code.

Data validation

Validate submitted location codes before saving address data in your application.

Implementation Examples

JavaScript

const BASE_URL = 'https://location-service-do.vercel.app'

async function getProvinces() {
  const res = await fetch(`${BASE_URL}/api/locations/provinces`)
  const json = await res.json()

  if (!json.status) {
    throw new Error(json.message)
  }

  return json.data
}

React Cascading Dropdown

import { useEffect, useState } from 'react'

const BASE_URL = 'https://location-service-do.vercel.app'

async function getJSON(path) {
  const res = await fetch(`${BASE_URL}${path}`)
  const json = await res.json()
  if (!json.status) throw new Error(json.message)
  return json.data
}

export function AddressSelect() {
  const [provinces, setProvinces] = useState([])
  const [regencies, setRegencies] = useState([])
  const [provinceCode, setProvinceCode] = useState('')

  useEffect(() => {
    getJSON('/api/locations/provinces').then(setProvinces)
  }, [])

  useEffect(() => {
    if (!provinceCode) {
      setRegencies([])
      return
    }

    getJSON(`/api/locations/regencies?province_code=${provinceCode}`)
      .then(setRegencies)
  }, [provinceCode])

  return (
    <>
      <select onChange={(event) => setProvinceCode(event.target.value)}>
        <option value="">Select province</option>
        {provinces.map((item) => (
          <option key={item.full_code} value={item.full_code}>
            {item.name}
          </option>
        ))}
      </select>

      <select disabled={!provinceCode}>
        <option value="">Select regency/city</option>
        {regencies.map((item) => (
          <option key={item.full_code} value={item.full_code}>
            {item.name}
          </option>
        ))}
      </select>
    </>
  )
}

Go

package main

import (
	"encoding/json"
	"fmt"
	"net/http"
)

type Response struct {
	Status bool `json:"status"`
	Data   []struct {
		Code     string `json:"code"`
		FullCode string `json:"full_code"`
		Name     string `json:"name"`
		Level    string `json:"level"`
	} `json:"data"`
}

func main() {
	res, err := http.Get("https://location-service-do.vercel.app/api/locations/provinces")
	if err != nil {
		panic(err)
	}
	defer res.Body.Close()

	var payload Response
	if err := json.NewDecoder(res.Body).Decode(&payload); err != nil {
		panic(err)
	}

	for _, item := range payload.Data {
		fmt.Println(item.FullCode, item.Name)
	}
}

Endpoints

GET /healthz

Check API availability.

GET /api/locations/stats

Get total location counts. Supports province_code, regency_code, and district_code for scoped counts.

GET /api/locations/provinces

Get all provinces.

GET /api/locations/regencies?province_code=11

Get regencies/cities by province.

GET /api/locations/districts?regency_code=11.01

Get districts by regency/city.

GET /api/locations/villages?district_code=11.01.01

Get villages by district.

GET /api/locations/search?q=aceh&limit=25

Search locations by name.

Code Format

By default, child endpoints return full administrative codes. Add code_format=short when your UI only needs child-level codes.

Level Full code Short code
Province 11 11
Regency/city 11.01 01
District 11.01.01 01
Village 11.01.01.2001 2001

Implementation Checklist

  • Use full_code as the saved location identifier.
  • Cache API responses in your frontend state to avoid duplicate requests.
  • Reset child dropdowns when a parent selection changes.
  • Show loading and empty states for each dropdown.
  • Use code_format=short only when your UI needs short child codes.
