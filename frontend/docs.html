Documentation

API wilayah Indonesia for address forms and location search.

Use this API to load provinces, regencies/cities, districts, and villages in your application. Start from provinces, then request the next level using the selected parent code.

Base URL https://location-service-do.vercel.app

Start Here

1

Load provinces

Use this on page load for your first dropdown.

GET /api/locations/provinces
2

Load child areas

When user selects a province, use its code to load regencies/cities. Repeat for districts and villages.

GET /api/locations/regencies?province_code=11
3

Save stable values

Store full_code and name in your system when you need reliable address references.

Endpoints

All endpoints return JSON. Add code_format=short when you only need child-level codes.

Health check GET /healthz
Total and scoped counts GET /api/locations/stats Optional: province_code, regency_code, district_code
Provinces GET /api/locations/provinces
Regencies / cities GET /api/locations/regencies?province_code=11
Districts GET /api/locations/districts?regency_code=11.01
Villages GET /api/locations/villages?district_code=11.01.01
Search by name GET /api/locations/search?q=aceh&limit=25

Response Format

Successful responses use the same envelope. Data can be an object or an array depending on endpoint.

{
  "log_id": "019ab0f0-c8ec-7d25-9f62-2f23d92fcda3",
  "code": 200,
  "status": true,
  "message": "Success",
  "data": [
    {
      "code": "11",
      "full_code": "11",
      "name": "Aceh",
      "level": "province"
    }
  ]
}

Implementation Examples

JavaScript fetch

const BASE_URL = 'https://location-service-do.vercel.app'

async function getProvinces() {
  const response = await fetch(`${BASE_URL}/api/locations/provinces`)
  const json = await response.json()

  if (!json.status) {
    throw new Error(json.message)
  }

  return json.data
}

Address dropdown flow

async function getRegencies(provinceCode) {
  return getJSON(`/api/locations/regencies?province_code=${provinceCode}`)
}

async function getDistricts(regencyCode) {
  return getJSON(`/api/locations/districts?regency_code=${regencyCode}`)
}

async function getVillages(districtCode) {
  return getJSON(`/api/locations/villages?district_code=${districtCode}`)
}

React example

import { useEffect, useState } from 'react'

const BASE_URL = 'https://location-service-do.vercel.app'

async function getJSON(path) {
  const response = await fetch(`${BASE_URL}${path}`)
  const json = await response.json()
  if (!json.status) throw new Error(json.message)
  return json.data
}

export function ProvinceSelect() {
  const [provinces, setProvinces] = useState([])

  useEffect(() => {
    getJSON('/api/locations/provinces').then(setProvinces)
  }, [])

  return (
    <select>
      <option value="">Select province</option>
      {provinces.map((item) => (
        <option key={item.full_code} value={item.full_code}>
          {item.name}
        </option>
      ))}
    </select>
  )
}
